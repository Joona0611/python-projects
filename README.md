# 🐍 Python Projects

## 📁 Salary Tracker

A project built with Python OOP (Object Oriented Programming).

### 📚 Concepts Learned:
- Getters and Setters
- Properties (`@property`)
- Class attributes
- Validation (TypeError, ValueError)
- `__str__` and `__repr__` methods

### 💡 About the Project:
An `Employee` class was created to manage an employee's
name, position, and salary.

### ✅ Features:
- Create an employee
- Promote to a higher level (demotion is not allowed)
- Update salary (cannot go below minimum salary)
- All values are validated
